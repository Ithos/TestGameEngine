Detailed Description

Material that defines color by its ambient, diffuse specular and emissive components. Also defines a global alpha value to stablish the transparency of the material.

Parameters
multiShadingInterfaceInterface to an object that implements custom shading for this material
ambientAmbient rgb color component of the material. Each color goes from 0.0 to 1.0
diffuseDiffuse rgb color component of the material. Each color goes from 0.0 to 1.0
specularSpecular rgb color component of the material. Each color goes from 0.0 to 1.0
emissiveEmissive rgb color component of the material. Each color goes from 0.0 to 1.0
globalAlphaValueMain alpha value for the whole object, clamps to 0.0 to 1.0 range. This value gets multiplied to every fragment color alpha.
shininessShininess component. Has to be > 0 if invalid takes the value 0.0001. Equation: spec contribution = cos(alpha) ^ shininess. If shininess is <= 0 it is set to 0.001 to avoid errors in the shaders.
translucentIndicates whether the material casts a translucent shadow or not
