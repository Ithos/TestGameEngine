Parameters
customShadingInterface to an object that implements custom shadow shading for this material
ambientTexDirKey to a texture that will be used as ambient color
diffuseTexDirKey to a texture that will be used as diffuse color
specularTexDirKey to a texture that will be used as specular color
emissiveTexDirKey to a texture that will be used as emissive color
thresholdValueMin alpha value below which the fragment is dropped completely. This value is clamped to the range [0, 1]
globalAlphaValueMain alpha value for the whole object, clamps to 0.0 to 1.0 range. This value gets multiplied to every fragment color alpha.
shininessShininess component. Has to be > 0 if invalid takes the value 0.0001. Equation: spec contribution = cos(alpha) ^ shininessIf shininess is <= 0 it is set to 0.001 to avoid errors in the shaders.
translucentIndicates whether the material casts a translucent shadow or not
