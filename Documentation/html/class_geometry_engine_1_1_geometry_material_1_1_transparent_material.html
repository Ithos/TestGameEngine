Detailed Description

Class that represents a material that may contain transparencies. Implements CustomShadingInterface so child classes can implement their own shadowMap shaders.

customShadingInterface to an object that implements custom shadow shading for this material
thresholdValueMin alpha value below which the fragment is dropped completely. This value is clamped to the range [0, 1]
globalAlphaValueGlobal alpha value to be applied to the whole model. This value is multiplied by other posible alphas. This value is clamped to the range [0, 1]
shininessShininess component. Equation: spec contribution = cos(alpha) ^ shininessIf shininess is <= 0 it is set to 0.001 to avoid errors in the shaders.
