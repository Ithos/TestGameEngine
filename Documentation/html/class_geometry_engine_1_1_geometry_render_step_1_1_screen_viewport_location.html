Detailed Description

Class that holds the locations and sizes for multiple screen viewports and their associated camera targets. QVector4D values must be in the [0;1] range, they indicate screen size percentages x,y -> viewport screen position z,w -> viewport width and height

void GeometryEngine::GeometryRenderStep::ScreenViewportLocation::generateViewportSizes ( int  viewportNumber,

protected

Clears the map passed as a parameters, then generates the sizes for the selected number of viewports and populates the map with them. This method will try to adjust the viewports as close as posible to a square matrix, aditional viewports will be added as aditional columns at certain rows from bottom to top. All viewport sizes will be adjusted so the full screen is filled.

Parameters
viewportNumberNumber of viewports in the screen
viewportMapViewport map to be cleared and filled usually this object viewport map
