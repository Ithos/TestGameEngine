maxLightAngleMax angle for the light. Attenuation will be applied based on angle deviation
attParamsAttenuation polynomial for the light
viewportViewport that will be used for the shadowmap calculation.
directionlight direction vector
boundingBoxGeomtry used to render the light. Light will be applied to every part of the scene that the geometry is drawn on top of. This class expects a 2D bounding geometry that will be rendered in front of the camera, applying the light like a filter.
diffuseDiffuse component of the light as an rgb vector. Each color goes from 0.0 to 1.0

posInitial position of the item
rotInitial rotaion of the item

scaleInitial scale to be applied to this item model
managerLight functionalities manager, defaults to nullptr.
parentPointer to this items parent item, nullptr if none.
